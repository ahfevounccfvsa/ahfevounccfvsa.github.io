玩docker，记录遇到的问题

Sep 8, 2017

###redis启动不了，遇到错误：Redis must be restarted after THP is disabled

错误提示中写明了：To fix this issue run the command 'echo never > /sys/kernel/mm/transparent_hugepage/enabled' as root, and add it to your /etc/rc.local in order to retain the setting after a reboot. Redis must be restarted after THP is disabled.

如果你在mac上开发的话，会发现找不到对应路径，其实应该在docker的vm中操作，我建了一个docker-machine，叫default，然后登陆进去改的。

  1. docker-machine ssh # 登陆进入默认的docker虚拟机，默认的是default，如果没有，请自己建立
  2. sudo -i # 表示切换为用户为root用户，否则无法sudu授权
  3. sudo echo never > /sys/kernel/mm/transparent_hugepage/enabled
  4. vi /etc/rc.local 把3对应的命令拷贝进去，然后:wq!保存退出
